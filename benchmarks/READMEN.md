# Benchmarks de ORYX

Este directorio contendrá pruebas comparativas entre:

- ORYX
- TOON
- JSON
- YAML

Las métricas incluirán:

- Consumo de tokens (prompts largos)
- Velocidad de parseo
- Tamaño del payload
- Precisión del LLM al responder usando cada formato

## Estado
Benchmarks serán añadidos en la versión **v0.2**.
