# ORYX — Especificación Oficial v0.1 (Draft)

## 1. Objetivo
ORYX es un formato optimizado para comunicación con modelos de lenguaje, reduciendo redundancia sintáctica y mejorando claridad estructural.

## 2. Principios del diseño
1. Densidad semántica máxima
2. Compatibilidad directa con JSON
3. Minimizar repetición de claves
4. Estructura clara para IA
5. Legible para humanos
6. Inferencia de tipos
7. Soporte dual: objetos y tablas
